# ⚔️ Eldrin: To-Do RPG

**Геймифицированный менеджер задач** — превращайте повседневные дела в RPG-приключение!

## 📋 О проекте

Eldrin: To-Do RPG — это веб-приложение, которое превращает обычный список дел в увлекательную ролевую игру. Выполняя задачи, вы получаете опыт, повышаете уровни персонажа и прокачиваете различные категории навыков.

### Основная идея

Стандартные менеджеры задач быстро надоедают — пользователи теряют мотивацию. Eldrin решает эту проблему через геймификацию: каждая задача становится квестом, а её выполнение приносит награду в виде опыта (XP).

### Ключевые особенности

- ✅ Создание задач с выбором категории и сложности
- 🎮 RPG-механика: опыт → уровни → прокачка
- 📊 5 категорий развития персонажа
- 🌙 Светлая и тёмная тема оформления
- 👥 Выбор класса персонажа
- 📁 Просмотр и восстановление выполненных задач
- 🔔 Красивые всплывающие уведомления

## 🎮 Как играть

### 1. Регистрация / Вход
- Создайте аккаунт или войдите с существующими данными
- Данные сохраняются в браузере (LocalStorage)

### 2. Создание задачи
- Нажмите «Создать новую задачу»
- Введите название задачи
- Выберите категорию (5 типов навыков)
- Выберите сложность (1-5) — от этого зависит награда

### 3. Выполнение задачи
- Нажмите ✅ на карточке задачи
- Вы получите опыт: `10 XP × сложность`
- При достижении порога уровень категории повышается

### 4. Прокачка персонажа
- Откройте вкладку «Профиль»
- Смотрите общий прогресс и уровни по категориям
- Выбирайте класс персонажа (кнопка с эмодзи в шапке)

### 5. Выполненные задачи
- Перейдите во вкладку «Выполненные»
- Можно вернуть задачу в активные или удалить навсегда

### 6. Смена темы
- Кнопка ☀️/🌙 в правом нижнем углу

## 📊 Игровая механика

### Формула опыта
XP за задачу = 10 × Сложность (1-5)


### Пороги уровней

| Уровень | Требуется XP | Порог следующего уровня |
|---------|--------------|------------------------|
| 1 → 2 | 100 XP | 100 XP |
| 2 → 3 | 150 XP | 150 XP |
| 3 → 4 | 225 XP | 225 XP |
| 4 → 5 | 338 XP | 338 XP |

> Порог увеличивается на 50% после каждого уровня

### Общий уровень персонажа
Сумма всех уровней по 5 категориям.  
Максимальный уровень не ограничен.

## 🔧 API и хранение данных

Проект использует **LocalStorage** браузера для хранения:

| Ключ | Содержание |
|------|------------|
| `eldrin_users` | Список пользователей |
| `eldrin_tasks_{userId}` | Задачи пользователя |
| `eldrin_stats_{userId}` | Статистика персонажа |
| `eldrin_character_{userId}` | Выбранный класс |

**Очистка данных:**
```javascript
localStorage.clear();

## 🐛 Возможные проблемы и решения

| Проблема | Решение |
|---------|--------------|
| Не меняется тема | Очистите localStorage и перезагрузите страницу |
| Пропал прогресс | Убедитесь, что вы вошли в тот же аккаунт |
| NaN в профиле | Очистите localStorage (F12 → Application → Clear) |
| Не работает авторизация | 	Проверьте, что введённый email уникален при регистрации |
